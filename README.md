# Heart-Disease-Predictor
# Heart Disease Prediction System

## 📌 Overview
This project is a Machine Learning-based system that predicts whether a person has a risk of heart disease based on medical attributes. It uses multiple classification algorithms and compares their performance to select the best model.

---

## 🚀 Features
- Data preprocessing and cleaning
- Handling categorical and numerical features
- Model training and comparison
- Multiple ML algorithms evaluation
- Real-time user input prediction system

---

## 🤖 Machine Learning Models Used
- Logistic Regression  
- Decision Tree Classifier  
- Support Vector Machine (SVM)  
- Naive Bayes Classifier  

---

## 📊 Workflow
1. Load dataset  
2. Data preprocessing (encoding & cleaning)  
3. Train-test split  
4. Feature scaling  
5. Train multiple models  
6. Compare accuracy  
7. Select best model  
8. Predict on user input  

---

## 🧠 Dataset Features
- Age  
- Sex  
- Chest Pain Type  
- Resting Blood Pressure  
- Cholesterol  
- Fasting Blood Sugar  
- Resting ECG  
- Maximum Heart Rate  
- Exercise Induced Angina  
- Oldpeak  
- ST Slope  
- Heart Disease (Target)

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  

---

## 📈 Output
The system predicts:
- ✅ Low risk of heart disease  
- ⚠️ High risk of heart disease
